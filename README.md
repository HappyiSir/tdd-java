## Training Puzzles

### Puzzle 1

我们考虑出租车计价问题，出租车的运价是每公里0.8元，八公里起会加收50%的每公里运价，起步价是两公里以内6块，停车等待时加收每分钟0.25元，最后计价的时候司机会四舍五入只收块块钱。

#### Tips

1. 注意被测对象的变化
2. 想简单，小步快跑
3. 不要忘记重构
